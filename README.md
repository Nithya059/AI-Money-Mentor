🚀 AI Money Mentor

📌 Problem

95% of Indians do not have access to structured financial planning.
Traditional financial advisors charge ₹25,000+ per year and primarily serve high-net-worth individuals (HNIs), leaving the majority without guidance.

---

💡 Solution

AI Money Mentor is an AI-powered personal finance assistant that helps users:

- Plan long-term wealth (FIRE)
- Optimize taxes
- Analyze investment portfolios
- Handle life-event financial decisions
- Receive autonomous financial action plans

---

⚙️ Features

🔥 FIRE Path Planner

- Multi-goal planning (Retirement, Education, House)
- Inflation-adjusted goal calculation
- Monthly SIP recommendations
- Asset allocation glidepath (Equity → Debt shift)
- Cashflow constraint handling

---

💯 Money Health Score

Evaluates financial wellness across:

- Emergency fund readiness
- Insurance coverage adequacy
- Investment diversification
- Debt health
- Tax efficiency
- Retirement readiness

---

🎯 Life Event Advisor

Handles financial impact of:

- Bonus
- Marriage
- Child
- Income/expense adjustments
- Risk profile updates
- Tax implications

---

💸 Tax Wizard

- Old vs New regime comparison
- HRA exemption calculation
- 80C, 80D, NPS deductions
- Home loan interest impact
- Step-by-step taxable income breakdown
- Tax-saving recommendations (ELSS, PPF, NPS)

---

📊 Portfolio X-Ray

- Mutual fund overlap detection
- Stock-level duplication analysis
- Tax-aware rebalancing suggestions
- XIRR calculation (cashflow-based)
- CAMS statement upload (basic parsing)

---

🤖 AI Financial Agent

Autonomous decision engine that:

- Detects financial problems
- Prioritizes actions (HIGH / MEDIUM / LOW)
- Resolves conflicts (Debt vs Investment)
- Maintains decision history
- Generates actionable plan

---

🧠 Tech Stack

- Python
- Streamlit (UI)
- Pandas & NumPy (data processing)
- Scikit-learn (ML model)
- yFinance (market data)
- pdfplumber (portfolio parsing)

---

🚀 Setup Instructions

1. Clone Repository

git clone https://github.com/Nithya059/AI-Money-Mentor.git
cd AI-Money-Mentor

2. Install Dependencies

pip install -r requirements.txt

3. Run Application

streamlit run app.py

---

🧪 Demo Flow

1. Enter personal financial details (income, expenses)
2. Add financial goals
3. View FIRE monthly investment plan
4. Analyze tax under both regimes
5. Upload portfolio or enter funds
